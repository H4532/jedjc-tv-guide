Sales publishing workflow

  1. Sign in using the Sales team member's approved GitHub account.
  2. Upload the approved landscape photo using a lowercase file name with hyphens.
  3. Open the matching data.json editor.
  4. Copy the existing template item and update both English and Arabic content.
  5. Set the image field to images/your-photo-name.jpg.
  6. Set active to true, then select Commit changes.
  7. Wait approximately one to three minutes and review the live page.
Access requirement: each Sales team member must have an individual GitHub account and be added as a collaborator to H4532/jedjc-tv-guide. GitHub Pages itself cannot provide a separate Sales login or restrict access to only one folder.
