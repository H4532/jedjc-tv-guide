# Holiday Inn Jeddah Corniche TV Guide

A bilingual, mobile-friendly guest TV channel guide for Holiday Inn Jeddah Corniche. Guests can search by channel number or name, filter by category, and download the matching PDF guide.

## Live site

https://h4532.github.io/jedjc-tv-guide/

The public guide deploys automatically through GitHub Pages after each update to the `main` branch.

## Included files

- `index.html` — accessible page structure
- `styles.css` — responsive guest-facing design
- `script.js` — channel search and category filters
- `TV_Channel_Guide.pdf.b64` — source for the downloadable bilingual PDF
- `.github/workflows/pages.yml` — automatic GitHub Pages deployment
